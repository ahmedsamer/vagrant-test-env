Ubuntu(12, 14 and 16), CentOS(6,7) with the following names and port map

| Ubuntu12 | 2222 | 2280 | 22443 |
| Ubuntu12 | 3322 | 3380 | 33443 |
| Centos6  | 4422 | 4480 | 44443 |
| Centos7  | 5522 | 5580 | 55443 |
